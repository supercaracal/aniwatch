# Aniwatch

## What's this?

This is my weekly television schedule. It is updated every three months.

## Development

```
$ git clone https://github.com/supercaracal/aniwatch.git
```

```
$ bin/bundle install --path=.bundle
```

```
$ bin/rails db:setup
```

```
$ bin/rails s -b 0.0.0.0
```

```
$ bin/guard
```
